Website History

Coding began as early as September 2020, however I spent very little time on it, beginning the wiki and press sections (still a work in progress). Fast.io was considered in December where developpement gained some speed but then it stopped being a thing so oopsie daisies I had to find something else, which led me to GitHub. On August 12th 2021 the website has been launched in its first version, hosting an index, the about page, an exploring portal with four sections, and a works portal which currently doesn't host anything. However, at the current moment, some unfinished components can be hidden in the website, those will be available soon. Over time redesigns have come, the website has become more pragmatic (for lack of a better word). Changes are still expected to come bit by bit, depends on my energy and time.
